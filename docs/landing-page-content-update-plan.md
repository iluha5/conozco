# Обновление контента главной страницы

## Цель

Полностью заменить текущий функционал и вид главной страницы на новый современный лендинг для зарегистрированных и незарегистрированных пользователей с тремя вариантами текста на трех языках. Переименовать сайт в "conozco" и добавить современные анимации при прокрутке.

## Структура обновления

### Файлы для изменения

- `app/page.tsx` - главная страница (полная замена всего содержимого, удаление всех существующих виджетов и карточек)
- `components/Header.tsx` - обновление названия с "Flash Cards" на "conozco" и логотипа

### Компоненты для создания

- `components/landing/` - новая папка для компонентов лендинга
- `components/landing/LandingContent.tsx` - основной компонент, который выбирает язык и рендерит соответствующий компонент
- `components/landing/LandingContentRu.tsx` - компонент с русским текстом
- `components/landing/LandingContentEn.tsx` - компонент с английским текстом
- `components/landing/LandingContentEs.tsx` - компонент с испанским текстом
- `components/landing/WorkflowDiagram.tsx` - компонент циклической диаграммы флоу работы
- `components/landing/LanguageLearningWarning.tsx` - блок с предупреждением о дополнительных инструментах
- `components/landing/RecentTrainings.tsx` - блок последних тренировок (для зарегистрированных)
- `components/landing/ActiveTraining.tsx` - блок текущей тренировки (заглушка)
- `components/landing/RegistrationCallToAction.tsx` - призыв к регистрации (для незарегистрированных)
- `components/landing/ScrollToTop.tsx` - компонент стрелки вверх для возврата наверх страницы
- `components/landing/ScrollAnimation.tsx` - обертка для анимаций при прокрутке
- `components/landing/landing-content.ts` - файл с тремя вариантами текстов на трех языках (структура по ключам: 'ru', 'en', 'es')
- `components/landing/useLandingLanguage.ts` - хук для работы с выбором языка (простой механизм по ключу языка)

## Три варианта текста

### Вариант 1: "Дружелюбный и мотивирующий"

- **Тон**: Энтузиазм, поддержка, дружелюбие
- **Акцент**: На прогрессе и достижениях
- **CTA**: Мотивирующие призывы к действию

### Вариант 2: "Профессиональный с элементами дружелюбия"

- **Тон**: Баланс между профессионализмом и доступностью
- **Акцент**: На эффективности и структурированности
- **CTA**: Четкие инструкции и преимущества

### Вариант 3: "Минималистичный и современный"

- **Тон**: Лаконичность, современность, ясность
- **Акцент**: На простоте использования
- **CTA**: Прямые и понятные призывы

### 1. Hero секция

- Заголовок и подзаголовок на трех языках
- Упоминание что на сайте можно изучать английский и испанский языки

- Анимация появления при загрузке страницы

- Рекомендация дополнительных инструментов (грамматика, письмо, разговорная речь)
- Формулировка на трех языках

### 3. Блок флоу работы (с циклической диаграммой)

- **Этап 1**: Поиск и добавление слов
- Через тренажеры по группам: `/training/list#tests`
- Ручное добавление: `/words`
- **Этап 2**: Тренировка новых слов
- Страница: `/training/list#new`
- **Этап 3**: Закрепление изученных слов
- Страница: `/training/list#learned`
- Циклическая диаграмма с визуализацией процесса и ссылками

### 4. Секция возможностей сервиса

- 6 этапов тренировки
- Система флеш-карточек
- Группы слов и тесты

- **Явное упоминание**: На текущий момент на сайте возможно изучение английского и испанского языков (указать в тексте на трех языках)

### 5. Блок для зарегистрированных пользователей

- **Последние запущенные тренировки**: Блок высотой примерно в одну карточку со страницы `/training/list` (стиль как у `TrainingModeCard`)

- Пока можно показать заглушку или пустое состояние

- **Текущая запущенная тренировка**: Блок для отображения активной тренировки
- Пока без реализации (заглушка)
- Показывать только если `session` существует

### 6. Блок для незарегистрированных пользователей

- **Призыв к регистрации**: Ненавязчивый блок с преимуществами регистрации
- Сохранение слов и прогресса
- Доступ к тренажерам на основе своих слов
- Указать что абсолютно бесплатно для всего функционала сайта
- Показывать только если `session` отсутствует

## Технические детали

### Мультиязычность и структура текстов

- **Важно**: Полноценный функционал локализации НЕ добавлять (будет добавлен позже для всего сайта)
- **Структура**: Разбить текст на три отдельных компонента по языкам:
- `LandingContentRu.tsx` - русский текст
- `LandingContentEn.tsx` - английский текст
- `LandingContentEs.tsx` - испанский текст
- **Механизм выбора языка**: Создать простой хук `useLandingLanguage.ts`:
- Работает с ключами языков: 'ru', 'en', 'es'
- Хранит выбранный язык в localStorage (ключ: `landing_language`)
- Возвращает текущий язык и функцию для его изменения
- Функционал вычисления языка пользователя НЕ обновлять (будет доработан позже)
- **Структура данных**: Файл `landing-content.ts` содержит три варианта текстов:
- Структура: `{ variant: { lang: { section: string } } }`
- Варианты: `variant1`, `variant2`, `variant3`
- Языки: `ru`, `en`, `es`
- Секции: `hero`, `warning`, `workflow`, `features`, `registration`
- **Подгрузка текстов**: Тексты подгружаются по ключу языка отдельно от будущего функционала локализации
- **Компонент-роутер**: `LandingContent.tsx` использует хук `useLandingLanguage` и рендерит соответствующий языковой компонент

### Диаграмма флоу

- Интерактивные ссылки на соответствующие страницы
- Адаптивный дизайн для мобильных устройств
- Яркие цвета и анимации для привлечения внимания
