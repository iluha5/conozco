### 3. Режим "Составление предложений" - подход реализации (КРИТИЧНО)

**Вопрос:** Как реализовать режим `learned-sentences` для изученных слов?

**Проблема:** Существующий `useTrainingData` загружает только `NOT_LEARNED` слова:

```typescript
const data = await trainingApi.fetchWords('NOT_LEARNED');
```

**Варианты решения:**

#### **Вариант A: Параметризация существующего flow** (рекомендуется)

**Изменения:**

1. Добавить параметр в `useTrainingData`:

    ```typescript
    export function useTrainingData(
        settingsLoaded: boolean,
        selectionLoaded: boolean,
        onWordsLoaded: (words: Word[]) => void,
        onLoadingChange: (loading: boolean) => void,
        wordStatus?: 'LEARNED' | 'NOT_LEARNED', // новый параметр
    );
    ```

2. Сохранять статус в `sessionStorage`:

    ```typescript
    sessionStorage.setItem(STORAGE_KEYS.TRAINING_WORD_SOURCE, 'LEARNED');
    ```

3. Читать в training page:
    ```typescript
    const wordSource = sessionStorage.getItem(
        STORAGE_KEYS.TRAINING_WORD_SOURCE,
    );
    const status = wordSource === 'LEARNED' ? 'LEARNED' : 'NOT_LEARNED';
    ```

✅ **Плюсы:**

- Минимальные изменения
- Переиспользование логики
- Единая точка входа

⚠️ **Минусы:**

- Изменение существующего flow
- Нужно тестировать регрессии

#### **Вариант B: Отдельный компонент**

Создать `LearnedWordsTrainingPage` с копией логики, но для изученных слов.

✅ **Плюсы:**

- Изоляция изменений
- Нет риска регрессий

❌ **Минусы:**

- Дублирование кода
- Сложность поддержки

**Рекомендация:** Вариант A - параметризация

### 4. Поведение при отсутствии группы A1 (Edge case)

**Вопрос:** Что делать, если группа A1 (id=4) не найдена или недоступна пользователю?

**Сценарии:**

1. Группа существует, но не активирована пользователем
2. Группа удалена из БД
3. У пользователя другой язык (группа для испанского, а у него английский)

**Варианты:**

- **a) Показать toast с инструкцией** (рекомендуется)

    ```typescript
    toast({
        title: 'Группа A1 недоступна',
        description: 'Добавьте группу "A1: All words" в разделе "Группы слов"',
        variant: 'destructive',
    });
    ```

- **b) Скрыть режимы A1**

    ```typescript
    const modesWithA1Check = LEARNED_TRAINING_MODES.filter(mode => {
        if (mode.groupName) return isGroupAvailable(mode.groupName);
        return true;
    });
    ```

- **c) Disable кнопки режимов A1**
    ```jsx
    <TrainingModeCard
        disabled={!isA1Available}
        tooltip="Группа A1 недоступна"
    />
    ```

**Рекомендация:** Комбинация (a) + (c) - показать disabled с tooltip, при клике - toast

### 5. Минимальное количество слов для режимов

**Вопрос:** Что делать, если изученных слов меньше требуемого?

**Сценарии:**

- Быстрая проверка требует 10, но есть только 5
- Sentences требует 5, но есть только 2
- A1 тест требует 10, но в группе только 7

**Варианты:**

- **a) Запустить с доступными** (рекомендуется)

    ```typescript
    if (selectedWordsList.length < config.wordCount) {
        toast({
            title: 'Недостаточно слов',
            description: `Требуется ${config.wordCount}, доступно ${selectedWordsList.length}. Тренировка будет запущена с доступными словами.`,
        });
        // Продолжаем с тем что есть
    }
    ```

- **b) Блокировать запуск**
    ```typescript
    if (selectedWordsList.length < config.wordCount) {
        toast({ title: 'Недостаточно слов', variant: 'destructive' });
        return { success: false, noWords: true };
    }
    ```

**Рекомендация:** Вариант (a) для лучшего UX

### 6. Настройки для режима "Составление предложений"

**Вопрос:** Какие именно настройки использовать для этапа 5?

**Текущие в плане:**

```typescript
{
    enabledStages: [5],
    wordCount: 5,
    settings: {
        stage5: { sentencesPerWord: 4 },
    },
}
```

**Уточнить:**

- Сколько предложений на слово? (в плане 4)
- Нужен ли stage 1 (просмотр) перед этапом 5?
- Порядок слов: sequential или random?

**Варианты:**

- **a) Только stage 5, 4 предложения**

    ```typescript
    enabledStages: [5],
    settings: { stage5: { sentencesPerWord: 4 } }
    ```

- **b) Stage 1 + Stage 5 (с просмотром)**
    ```typescript
    enabledStages: [1, 5],
    settings: {
        stage1: { showExamples: false },
        stage5: { sentencesPerWord: 4 }
    }
    ```

**Рекомендация:** Вариант (a) - только stage 5
