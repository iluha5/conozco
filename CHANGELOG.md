# Changelog

Все значимые изменения в проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект следует [Semantic Versioning](https://semver.org/lang/ru/).

## [0.1.0] - 2024-10-24

### Добавлено
- Начальная версия приложения Flash Cards
- Поддержка английского и испанского языков
- Управление словарём (добавление, удаление, просмотр)
- Система переводов с кешированием
- Моки для Google Translate API
- 4 этапа тренировки:
  - Этап 1: Просмотр слова с озвучкой
  - Этап 2: Выбор правильного перевода
  - Этап 3: Сопоставление слов
  - Этап 4: Составление слова из букв
- Фильтрация слов по языку и статусу
- Статус слов (выучено/не выучено)
- База данных PostgreSQL с Prisma ORM
- Docker и Docker Compose конфигурация
- UI на базе shadcn/ui
- Toast уведомления
- Статистика тренировок
- Примеры использования слов

### Технические детали
- Next.js 14 с App Router
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL 16
- Docker контейнеризация
- RESTful API

### Документация
- README.md с инструкциями по запуску
- USAGE.md с руководством пользователя
- ARCHITECTURE.md с описанием архитектуры
- Комментарии в коде

## [0.1.1] - 2024-10-24

### Исправлено
- Исправлена ошибка OpenSSL в Docker контейнере (Prisma требует OpenSSL)
- Добавлены системные зависимости `openssl` и `libc6-compat` в Dockerfile
- Удален устаревший атрибут `version` из docker-compose.yml
- Исправлена ошибка `useState only works in Client Components`
- Добавлена директива `'use client'` во все клиентские компоненты (8 файлов)

### Изменено
- Обновлен синтаксис с `docker-compose` на `docker compose` во всей документации
- Обновлены команды в README.md, QUICKSTART.md, USAGE.md, ARCHITECTURE.md
- Обновлен скрипт start.sh

### Добавлено
- Документ FIXES.md с описанием исправлений
