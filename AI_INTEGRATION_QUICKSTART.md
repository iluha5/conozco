## 🔧 Как работает кнопка "Найти"

### Кнопка активна (можно нажать):

- ✅ Поле поиска НЕ пустое
- ✅ В базе НЕТ точного совпадения
- ✅ Поиск НЕ выполняется

### Кнопка неактивна (серая):

- ❌ Поле поиска пустое
- ❌ Слово уже есть в базе (точное совпадение)
- ❌ Поиск уже выполняется

### Процесс поиска:

1. **Перевод** через MyMemory API (es/en → ru)
    - Таймаут: 10 секунд
    - Если не удалось - показывается ошибка
2. **Примеры** через Tatoeba API (параллельно)
    - Первая попытка сразу
    - Если не нашлись - до 3 попыток в фоне
    - Интервал между попытками: 2 секунды

3. **Сохранение** в базу данных:
    - BaseWord (если не существует)
    - WordTranslation (основной + альтернативы)
    - WordExample (до 5 примеров)
    - Word (добавление в словарь пользователя)

## 📊 Логи API запросов

Все запросы логируются в таблицу `ApiRequestLog`:

### Просмотр логов в БД:

```sql
-- Последние 20 запросов
SELECT
  "serviceName",
  "requestType",
  "statusCode",
  "duration",
  "createdAt",
  "errorMessage"
FROM "ApiRequestLog"
ORDER BY "createdAt" DESC
LIMIT 20;

-- Статистика по сервисам
SELECT
  "serviceName",
  COUNT(*) as total_requests,
  AVG("duration") as avg_duration_ms,
  COUNT(CASE WHEN "statusCode" >= 200 AND "statusCode" < 300 THEN 1 END) as success,
  COUNT(CASE WHEN "errorMessage" IS NOT NULL THEN 1 END) as errors
FROM "ApiRequestLog"
GROUP BY "serviceName";
```

## ⚠️ Ограничения

### MyMemory API:

- Лимит: 10,000 символов/день (бесплатно)
- Качество: хорошее для основных языков
- Скорость: ~1-2 секунды

### Tatoeba API:

- Лимит: нет (полностью бесплатно)
- Качество: отличное (реальные примеры)
- Скорость: ~1-2 секунды
- Примеры могут отсутствовать для редких слов

## 🐛 Возможные проблемы

### Проблема: Кнопка "Найти" не активируется

**Решение:**

- Проверьте, что слова нет в базе (попробуйте ввести редкое слово)
- Убедитесь, что поле поиска не пустое

### Проблема: "Не удалось найти перевод"

**Решение:**

- Проверьте интернет соединение
- MyMemory API может быть временно недоступен
- Попробуйте через 1-2 минуты

### Проблема: Слово добавилось без примеров

**Решение:**

- Это нормально! Не для всех слов есть примеры в Tatoeba
- Система делает до 3 попыток найти примеры в фоне
- Можете добавить примеры вручную позже
