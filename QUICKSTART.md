# ⚡ Быстрый старт Flash Cards

## 🚀 Запуск за 3 шага

### 1. Убедитесь, что Docker установлен

```bash
docker --version
docker-compose --version
```

Если Docker не установлен, скачайте с [docker.com](https://www.docker.com/products/docker-desktop)

### 2. Запустите приложение

**Linux/MacOS:**
```bash
chmod +x start.sh
./start.sh
```

**Windows:**
```bash
docker compose up --build
```

**Альтернативный способ (любая ОС):**
```bash
docker compose up --build -d
```

### 3. Откройте браузер

Перейдите по адресу: **http://localhost:3000**

---

## 📚 Первые шаги

### Добавьте первое слово

1. На главной странице нажмите **"Управление словами"**
2. Нажмите **"Добавить слово"** (кнопка с плюсом)
3. Выберите язык (Английский или Испанский)
4. Введите слово, например `hello`
5. Нажмите **"Поиск"**
6. Выберите один из предложенных переводов
7. Нажмите **"Добавить слово"**

### Попробуйте тренировки

1. Добавьте несколько слов (минимум 4 для полноценной тренировки)
2. Вернитесь на главную страницу
3. Нажмите **"Начать тренировку"**
4. Выберите этап тренировки (1-4)
5. Следуйте инструкциям на экране

## 🎯 Доступные слова для тестирования

Система включает моки для следующих слов:

### 🇬🇧 Английский
- `hello` - привет/здравствуйте/алло
- `book` - книга/бронировать/записывать
- `run` - бежать/запускать/управлять/течь
- `water` - вода/поливать
- `light` - свет/легкий/светлый/зажигать

### 🇪🇸 Испанский
- `hola` - привет/здравствуйте
- `casa` - дом/жилище
- `tiempo` - время/погода/период
- `amor` - любовь/милый

Вы также можете вводить любые другие слова - система предложит базовый перевод.

## 🎓 4 этапа тренировки

### Этап 1: Просмотр и запоминание
- Просмотр слова с автоматической озвучкой
- Показ перевода по клику
- Идеально для начального знакомства со словами

### Этап 2: Выбор правильного перевода
- Показывается иностранное слово
- Нужно выбрать правильный перевод из 4 вариантов
- Отслеживание правильных ответов

### Этап 3: Сопоставление
- Соедините иностранные слова с их переводами
- Тренировка группами по 4 слова
- Развивает ассоциативную память

### Этап 4: Составление слова
- Показывается русский перевод
- Нужно составить иностранное слово из букв
- Развивает правописание

## 📊 Управление словами

### Фильтры
- **По языку**: Все / Английский / Испанский
- **По статусу**: Все / Не выучено / Выучено

### Действия
- **Отметить как выученное** - переместить в список выученных слов
- **Удалить** - удалить слово из словаря
- **Статистика** - количество слов по категориям

## ❓ Решение проблем

### Порт 3000 занят
Измените в `docker-compose.yml`:
```yaml
ports:
  - "3001:3000"  # Используйте другой порт
```

### Порт 5432 (PostgreSQL) занят
Измените в `docker-compose.yml`:
```yaml
postgres:
  ports:
    - "5433:5432"  # Используйте другой порт
```

И обновите `.env`:
```
DATABASE_URL="postgresql://flashcards:flashcards_password@localhost:5433/flashcards"
```

### База данных не создаётся
```bash
# Полная очистка и перезапуск
docker compose down -v
docker system prune -f
docker compose up --build
```

### Озвучка не работает
Озвучка использует Web Speech API и работает в:
- ✅ Chrome/Edge
- ✅ Safari
- ⚠️ Firefox (ограниченная поддержка)
